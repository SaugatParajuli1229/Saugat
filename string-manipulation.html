String Manipulation Toolkit

C Programming

Overview

A comprehensive toolkit providing various string manipulation utilities. This project demonstrates essential string handling techniques and array manipulation in C, including common algorithms for string processing.

Features

  • Palindrome checker - detects if a string reads the same forwards and backwards
  • Word counter - counts words in a given string
  • Character reversal - reverses a string character by character
  • Case conversion - converts between uppercase and lowercase
  • String length calculator
  • Character frequency analyzer
  • Substring finder
  • Whitespace trimmer

Technologies Used

C String Functions Arrays Character Processing

Code Examples

// Palindrome Checker
int isPalindrome(char str[]) {
    int left = 0, right = strlen(str) - 1;
    while(left < right) {
        if(str[left] != str[right])
            return 0;
        left++;
        right--;
    }
    return 1;
}

// String Reversal
void reverseString(char str[]) {
    int n = strlen(str);
    for(int i = 0; i < n/2; i++) {
        char temp = str[i];
        str[i] = str[n-1-i];
        str[n-1-i] = temp;
    }
}

// Word Counter
int countWords(char str[]) {
    int count = 0;
    int inWord = 0;
    for(int i = 0; str[i] != '\0'; i++) {
        if(str[i] == ' ' || str[i] == '\t' || str[i] == '\n') {
            inWord = 0;
        } else if(!inWord) {
            count++;
            inWord = 1;
        }
    }
    return count;
}
