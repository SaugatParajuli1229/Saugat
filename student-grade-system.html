Student Grade Management System

C Programming

Overview

A console-based program to manage student records including name, roll number, and grades. This project demonstrates fundamental C programming concepts including data structures, file I/O, and function design.

Features

  • Add new student records with name, roll number, and grades
  • Display all students or search for specific students
  • Calculate average grade for individual students
  • Calculate class average grade
  • Persistent storage using file I/O
  • Data validation and error handling

Technologies Used

C Arrays Structures File I/O Functions

Code Example

typedef struct {
    int rollNo;
    char name[50];
    float grades[5];
    float average;
} Student;

void addStudent(Student *students, int *count) {
    printf("Enter student name: ");
    scanf("%s", students[*count].name);
    printf("Enter roll number: ");
    scanf("%d", &students[*count].rollNo);
    printf("Enter 5 grades: ");
    
    float sum = 0;
    for(int i = 0; i < 5; i++) {
        scanf("%f", &students[*count].grades[i]);
        sum += students[*count].grades[i];
    }
    
    students[*count].average = sum / 5;
    (*count)++;
}

Learning Outcomes

  • Understanding of C structures and arrays
  • Implementation of file I/O operations
  • Function design and parameter passing
  • Menu-driven program development
  • Data persistence and retrieval
