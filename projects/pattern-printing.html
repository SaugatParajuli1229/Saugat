Pattern Printing Programs

C Programming

Overview

A comprehensive collection of programs demonstrating nested loop logic through various pattern generation exercises. These programs range from simple triangles to complex pyramids and ASCII art, making it an excellent resource for understanding loop iteration and conditional logic.

Patterns Included

  • Triangular patterns (right, left, equilateral)
  • Pyramid patterns (numbers, stars, characters)
  • Diamond patterns with various fills
  • Hollow shapes (rectangles, diamonds)
  • ASCII art designs
  • Floyd's triangle
  • Pascal's triangle
  • Multiplication tables in pattern format

Technologies Used

C Nested Loops Pattern Logic Character Output

Code Examples

// Right Triangle Pattern
void rightTriangle(int n) {
    for(int i = 1; i <= n; i++) {
        for(int j = 1; j <= i; j++) {
            printf("* ");
        }
        printf("\n");
    }
}

// Diamond Pattern
void diamond(int n) {
    for(int i = 1; i <= n; i++) {
        for(int j = 1; j <= n - i; j++) {
            printf(" ");
        }
        for(int j = 1; j <= 2*i - 1; j++) {
            printf("*");
        }
        printf("\n");
    }
    for(int i = n - 1; i >= 1; i--) {
        for(int j = 1; j <= n - i; j++) {
            printf(" ");
        }
        for(int j = 1; j <= 2*i - 1; j++) {
            printf("*");
        }
        printf("\n");
    }
}
