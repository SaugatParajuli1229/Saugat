Fibonacci & Prime Number Generator

C Programming

Overview

A collection of mathematical programs that generate Fibonacci sequences and identify prime numbers up to a given limit. These projects demonstrate fundamental algorithm implementation and loop control structures in C.

Features

  • Fibonacci sequence generation with iterative approach
  • Recursive Fibonacci implementation
  • Prime number detection and validation
  • Generate all prime numbers up to N (Sieve of Eratosthenes)
  • Performance comparison between iterative and recursive approaches
  • User-friendly console interface

Technologies Used

C Loops Algorithms Recursion

Code Examples

// Fibonacci - Iterative Approach
void fibonacciIterative(int n) {
    int a = 0, b = 1;
    printf("Fibonacci Series: ");
    
    for(int i = 0; i < n; i++) {
        printf("%d ", a);
        int temp = a + b;
        a = b;
        b = temp;
    }
    printf("\n");
}

// Prime Number Checker
int isPrime(int num) {
    if(num <= 1) return 0;
    if(num == 2) return 1;
    if(num % 2 == 0) return 0;
    
    for(int i = 3; i * i <= num; i += 2) {
        if(num % i == 0) return 0;
    }
    return 1;
}
