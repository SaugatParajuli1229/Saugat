Personal Portfolio Website

Web Development

Overview

A responsive portfolio website showcasing web development fundamentals. This website serves as a comprehensive introduction to web development using HTML5 for semantic structure and CSS3 for modern styling, featuring multiple pages and smooth navigation.

Features

  • Multi-page structure (Home, About, Projects, Contact)
  • Responsive design that works on desktop, tablet, and mobile
  • Modern CSS3 styling with gradients and transitions
  • Navigation bar with active page highlighting
  • Project showcase grid with detailed project cards
  • Contact form with validation
  • Semantic HTML5 markup
  • Mobile-first design approach

Technologies Used

HTML5 CSS3 JavaScript Responsive Design

Key Code Highlights

/* Responsive Grid Layout */
.projects-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 2rem;
    padding: 2rem 0;

/* Mobile First Media Query */
@media (max-width: 768px) {
    .projects-grid {
        grid-template-columns: 1fr;
    }
    
    .menu {
        flex-wrap: wrap;
    }
}
