Number Guessing Game

C Programming

Overview

An interactive console game where users guess a randomly generated number within a specified range. The game provides hints and tracks the number of attempts, making it an excellent project for understanding random number generation and conditional logic.

Features

  • Random number generation within a user-defined range
  • Interactive guess-checking with hints (higher/lower)
  • Attempt counter to track guesses
  • Scoring system based on number of attempts
  • Replay functionality
  • Difficulty levels (Easy, Medium, Hard)
  • Input validation and error handling

Technologies Used

C Random Numbers Conditionals Loops

Code Example

void playGame() {
    int secretNumber = rand() % 100 + 1;
    int guess = 0;
    int attempts = 0;
    
    printf("Guess a number between 1 and 100\n");
    
    while(guess != secretNumber) {
        printf("Enter your guess: ");
        scanf("%d", &guess);
        attempts++;
        
        if(guess > secretNumber) {
            printf("Too high! Try again.\n");
        }
        else if(guess < secretNumber) {
            printf("Too low! Try again.\n");
        }
        else {
            printf("Correct! You guessed it in %d attempts.\n", attempts);
            calculateScore(attempts);
        }
    }
}

Learning Outcomes

  • Random number generation using rand() and srand()
  • Conditional statements and logical operators
  • Loop control (while, break statements)
  • User input validation
  • Game logic implementation
  • Difficulty scaling concepts
